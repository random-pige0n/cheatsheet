  1. HTML

  2. What is HTML?

HTML stands for HyperText MarkupLanguage. It's the easiest and first computer language you need to learn to be able to build websites.
In the table below you'll find a cheatsheet for getting started with HTML.

Name Tag Description
Head <head>

Is used to make the head of the HTML document, what goes here wont be seen on the screen.
Link <link> Here you can link CSS stylesheet or other links like fonts.
Title <title> The title of your page goes here, and this tag goes in the head. You can see it as the title in the browser tab.
Body <body> Here goes the body elements of the HTML like h1-h6, p, tables, forms etc.

  1. CSS

  2. What is CSS?

CSS stands for Cascading Style Sheets, it's like clothes for your web page, it adds color, images, style etc. With these two posibilities for what your website looks like are endless. Below are some cheatsheets to get an idea of how it works.

Name Attribute Description
Text align text-align: ; This attribute centers text, bestween the colon and semi-colon goes the value of the attribute. Here it can be center, left or right just like in microsoft word.
Display display: ; This can control how the boxes of elements are displayed, inline is default for p, meaning it takes only as much space as needed. Block for elements like h1-h6, means they start a new line and take the whole width. inline-block is like inline but width and height can be set
Padding padding: ; This adds space between the element and its border, to make it in one attribute, the values are padding: top left bottom right;
Margin margin: ; Adds space between the border and the edge of the page, same rules apply as for padding. also for both margin-direction can be used instead if only one value needs to be set.
